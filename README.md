# Cognita — AI Teaching Assistant

An offline-first mobile AI teaching assistant built for Pakistani CS students. 
Fine-tuned TinyLlama 1.1B on 5,030 CS teaching conversations using QLoRA. 
Deployed as a Flutter mobile app with live HuggingFace backend.

## Results
- Training loss reduced from 2.34 to 0.55
- 4-bit quantization via BitsAndBytesConfig
- 12+ model downloads on HuggingFace

## Live Links
- HuggingFace Space: https://huggingface.co/spaces/SyedaArisha/cognita
- Model: https://huggingface.co/SyedaArisha/cognita-tinyllama

## Tech Stack
Flutter, TinyLlama 1.1B, QLoRA, PEFT, BitsAndBytes, Gemini API, Firebase, HuggingFace
